MakeCpp+ Now Available!

发表于 2016-03-26   |  

What is MakeCpp+?

MakeCpp+ —— 一个小巧的代码模板生成器。

Made by Xeon Stin (hjf222289)

它可以完成以下功能：

  1. 根据输入文件名自动生成以该名称命名的 C++ 源代码　*.cpp 与空白读入文件 *.in
  2. 能够根据你的需求生成多种预置模板，如：
    1. 树状数组
    2. 线段树
    3. ST 表
    4. 并查集
    5. 高精度数读入以及加法
    6. 快速读入
    7. 快速幂
  3. 但需要注意的是，由于预置模板的特性，生成的代码可能需要针对实际需求进行一些改动。

How to Use MakeCpp+?

上图即为 MakeCpp+ 的操作界面，其中：

  1. 在上方的框中输入文件名。
  2. file_IO 即为开启文件读写，默认勾选。其默认读入文件为 name.in，输出文件为 name.out，其中 name 为框中输入的文件名。勾选后会一并生成 name.in 空白文件。
  3. Exit after Generation 即为生成后自动退出，默认勾选。若需要连续生成可去掉勾选。
  4. binary_index_tree 即为开启预置树状数组模板，其余同理。
  5. 配置完毕后，点击 Generate 按钮即可生成代码文件。

所有操作如以上所述。

仅寥寥几个选项，简单易用又实用，实乃居家旅行、杀人越货，懒人必备之工具是也。 —— Xeon Stin

  4. 然后就现学 C# 写了一个带 GUI 的文件读写生成器
  5. 为什么不再懒一点呢？
  6. 于是乎贴上了一大坨模板啦函数啦之类的
