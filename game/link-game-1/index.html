Welcome to Stage 0 of a link game made by hjf222289

How does this game work?

很简单。你只需要得到每个题目所对应的答案（通常是一个字符串），然后把像这样它输入到地址栏里：

currentsite/game/link-game-1/answer

其中 currentsite 为当前地址栏 /game/link-game-1 前的部分

answer 为本题答案。

然后你就可以进入下一题的页面了。

那么本题的答案是 the-game-is-on。
